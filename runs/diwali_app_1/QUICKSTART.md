# 🚀 Quick Start Guide - Diwali App

## Fastest Way to Run (3 Methods)

### Method 1: Double-Click (Easiest!) 🖱️
```
Just double-click on 'index.html' and it will open in your default browser!
```

### Method 2: Python Launcher (Recommended) 🐍
```bash
python3 launch.py
```
This will:
- Start a local web server
- Automatically open the app in your browser
- Show you the URL (http://localhost:8000)

### Method 3: Manual Server 🔧
```bash
# Using Python
python3 -m http.server 8000

# Then open: http://localhost:8000
```

## Troubleshooting

### Port Already in Use?
```bash
# Try a different port
python3 -m http.server 8080
# Then open: http://localhost:8080
```

### Python Not Found?
- Just double-click `index.html` instead!
- Or install Python from python.org

### Browser Issues?
- Use Chrome, Firefox, Safari, or Edge
- Make sure JavaScript is enabled
- Clear cache if needed (Ctrl+Shift+R or Cmd+Shift+R)
