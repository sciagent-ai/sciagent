# 🪔 Diwali App - Festival of Lights 🪔

A beautiful, interactive web application celebrating Diwali, the Festival of Lights!

## Features

✨ **Animated Diyas (Oil Lamps)** - Three beautiful animated oil lamps with flickering flames
🎆 **Fireworks Display** - Interactive fireworks that burst across the screen
💫 **Sparkle Effects** - Continuous sparkling particles in the background
🌺 **Rotating Rangoli** - Colorful rotating rangoli pattern
🎉 **Celebrate Button** - Click to trigger a spectacular fireworks show
📱 **Responsive Design** - Works beautifully on all devices

## How to Run

### Option 1: Direct Browser Opening
Simply open `index.html` in your web browser by double-clicking the file.

### Option 2: Using Python HTTP Server
```bash
# Navigate to the diwali_app_1 directory
cd runs/diwali_app_1

# Start a simple HTTP server (Python 3)
python3 -m http.server 8000

# Or with Python 2
python -m SimpleHTTPServer 8000
```

Then open your browser and go to: `http://localhost:8000`

### Option 3: Using Node.js HTTP Server
```bash
# Install http-server globally (if not already installed)
npm install -g http-server

# Navigate to the diwali_app_1 directory
cd runs/diwali_app_1

# Start the server
http-server -p 8000
```

Then open your browser and go to: `http://localhost:8000`

## Features Explained

### Animated Diyas
Three traditional oil lamps with realistic flickering flame animations that float gently.

### Fireworks
- Automatic fireworks display on page load
- Click the "Celebrate!" button for more fireworks
- Random fireworks appear every 5 seconds
- Multiple colors: orange, gold, pink, cyan, and more

### Sparkles
Continuous golden sparkles appear randomly across the screen, creating a magical atmosphere.

### Rangoli
A colorful, rotating rangoli pattern that speeds up when you hover over it.

## Customization

Feel free to customize the colors, animations, and messages in the code:
- Colors are defined in `style.css`
- Animation speeds can be adjusted in both CSS and JavaScript
- Messages can be changed in `index.html`
