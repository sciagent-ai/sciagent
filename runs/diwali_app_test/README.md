# 🪔 Diwali Celebration App

A simple, interactive Python application to celebrate Diwali - the Festival of Lights!

## Features

✨ **Interactive Menu System**
- Light virtual diyas (lamps)
- Display colorful rangoli patterns
- Watch animated fireworks
- Read beautiful Diwali wishes
- Full celebration mode with all features

🎨 **Visual Elements**
- Colorful ASCII art
- Animated diya lighting
- Beautiful rangoli designs
- Fireworks simulation
- Festive banners and decorations

## Installation

No external dependencies required! Uses only Python standard library.

### Requirements
- Python 3.6 or higher
- Terminal with ANSI color support (most modern terminals)

## Usage

### Run the Application

```bash
python diwali_app.py
```

Or make it executable:

```bash
chmod +x diwali_app.py
./diwali_app.py
```

### Menu Options

1. **🪔 Light a Diya** - Watch a beautiful diya lighting animation
2. **🎨 Display Rangoli** - See a colorful rangoli pattern
3. **🎆 Watch Fireworks** - Enjoy animated fireworks display
4. **💌 Read Diwali Wishes** - Read heartfelt Diwali greetings
5. **🎉 Full Celebration** - Experience all features in sequence
6. **🚪 Exit** - Exit the application

## Features Explained

### Diya Animation
A traditional oil lamp (diya) is displayed with flickering flame animation, symbolizing the victory of light over darkness.

### Rangoli Display
A beautiful, colorful rangoli pattern made with ASCII art characters, representing the traditional floor decorations made during Diwali.

### Fireworks Show
An animated fireworks display with random colors and positions, celebrating the festive spirit.

### Diwali Wishes
Heartfelt wishes and blessings for health, wealth, prosperity, and happiness.

## Technical Details

- **Language**: Python 3
- **Dependencies**: None (uses standard library only)
- **Terminal**: Requires ANSI color support
- **Platform**: Cross-platform (Linux, macOS, Windows with modern terminal)

```

├── Wishes display

```

## Customization

You can easily customize:
- Colors in the `COLORS` dictionary
- ASCII art patterns for diya and rangoli
- Wishes text
- Animation speeds (adjust `time.sleep()` values)
- Fireworks patterns and frequency

## About Diwali

Diwali (Deepavali) is one of the most important festivals in Indian culture, celebrating:
- The victory of light over darkness
- Good over evil
- Knowledge over ignorance

It's celebrated with:
- Lighting diyas and candles
- Creating rangoli designs
- Bursting fireworks
- Exchanging gifts and sweets
- Family gatherings and prayers
