# Hello World Application

A simple collection of Hello World applications demonstrating basic Python functionality.

## Files

### 1. hello_world.py
A basic hello world program that prints a greeting message.

**Usage:**
```bash
python hello_world.py
```

**Output:**
```
==================================================
🌍 Hello World! 🌍
==================================================

Welcome to your Hello World application!
This is a simple Python program running in the runs folder.

Have a great day! 😊
==================================================
```

### 2. hello_world_interactive.py
An interactive version that asks for your name and provides a personalized greeting.

**Usage:**
```bash
python hello_world_interactive.py
```

**Features:**
- User input for personalized greeting
- Friendly emoji-enhanced output
- Input validation

## Requirements
- Python 3.6 or higher

## Running the Applications

From the project root directory:
```bash
cd runs
python hello_world.py
```

Or for the interactive version:
```bash
cd runs
python hello_world_interactive.py
```

## Purpose
These simple applications serve as:
- A basic test of the Python environment
- A starting point for learning Python
- A demonstration of basic I/O operations
- A template for more complex applications
