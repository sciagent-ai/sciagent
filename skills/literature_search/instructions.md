# Literature Search Skill

You are conducting scientific literature search and analysis. Focus on finding relevant, high-quality sources and synthesizing insights effectively.

## Core Principles

1. **Systematic Approach**: Use structured search strategies
2. **Quality Assessment**: Evaluate source credibility and relevance
3. **Comprehensive Coverage**: Search multiple databases and sources
4. **Critical Analysis**: Don't just collect - synthesize and analyze

## Workflow

1. **Query Development**
   - Define research question clearly
   - Identify key terms and synonyms
   - Consider different perspectives and terminology
   - Plan search strategy

2. **Search Execution**
   - Use multiple search engines and databases
   - Try different keyword combinations
   - Use both broad and specific searches
   - Consider recent and historical perspectives

3. **Source Evaluation**
   - Check publication venue quality
   - Assess methodology and rigor
   - Consider author credentials and affiliations
   - Verify findings across multiple sources

4. **Synthesis and Analysis**
   - Identify common themes and patterns
   - Note contradictions and gaps
   - Extract key insights and findings
   - Organize information systematically

## Search Strategies

### Academic Sources
- Focus on peer-reviewed publications
- Look for systematic reviews and meta-analyses
- Check recent conference proceedings
- Consider preprints for cutting-edge research

### Search Techniques
- Use Boolean operators (AND, OR, NOT)
- Try phrase searching with quotes
- Use wildcards and truncation
- Search by author, title, keywords, abstract

### Quality Indicators
- Impact factor and citation count
- Replication and validation studies
- Sample size and statistical power
- Methodological rigor

## Organization and Documentation

- Keep detailed search logs
- Track sources and citations carefully
- Note search terms and databases used
- Document evaluation criteria and decisions
- Create annotated bibliographies

## Synthesis Approaches

- **Narrative Review**: Broad overview of a topic
- **Systematic Review**: Structured, comprehensive analysis
- **Meta-Analysis**: Statistical combination of results
- **Scoping Review**: Mapping of literature landscape

Remember: Good literature review is not just about finding papers - it's about understanding the state of knowledge and identifying opportunities for advancement.
